# PhotoSoft

A lightweight Paint.NET-inspired image editor built with HTML, CSS, and JavaScript. Currently in Alpha

## Features

- Brush Tool
- Eraser Tool
- Color Picker
- Brush Size Control
- Image Import
- PNG Export

## Roadmap

- Undo / Redo
- Fill Bucket
- Eyedropper
- Recolor Tool
- Magic Wand
- Layers

Early development 🚀
