# Changelog

All notable changes to PhotoSoft will be documented in this file.

---

## [0.2] - Foundation Release

Release Date: June 2026

### Added
- Modern user interface
- HTML5 Canvas rendering engine
- Brush Tool
- Eraser Tool
- Color Picker
- Adjustable Brush Size
- Image Import (.png, .jpg, .jpeg, .webp)
- PNG Export
- Tool Selection System
- Dark Theme UI
- Poppins Font Integration

### Fixed
- Canvas initialization issues
- JavaScript loading issues
- Image upload problems
- Tool activation bugs

### Notes
This is the first stable PhotoSoft Alpha release.
The primary goal of v0.2 was to establish a working foundation for future editing features.
